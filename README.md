Jeepay Plus 是基于开源版 Jeepay 搭建的聚合支付系统，我们对部分服务端代码进行了重构，并新增了以下功能：
- **进件管理**：完善商户入驻流程。
- **代理商系统**：支持多层级代理的高效管理。
- **商户通 APP**：为商户提供便捷的移动端管理工具。
- **展业宝 APP**：帮助代理商更好地拓展业务。

Jeepay Plus 历时 2 个月精心打磨，旨在打造一个真正可落地使用的聚合支付系统，为商业客户提供完整的支付解决方案。
